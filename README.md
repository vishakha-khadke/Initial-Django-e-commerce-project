# Django E-Commerce Website

This is a beginner-friendly Django-based e-commerce web application.

## Features
- Product listing
- Add to cart
- Cart management
- Checkout page
- Admin panel for product management

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite
- Git & GitHub

## How to Run
1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start server using:
   python manage.py runserver
